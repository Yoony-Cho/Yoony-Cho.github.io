## What is a HTTP?

**HTTP**는 **Hyper Text Transfer Protocol**의 약자로, 인터넷에서 데이터를 주고받을 수 있는 프로토콜입니다.

HTTP는 웹 페이지, 이미지, 동영상 등을 주고받는 데 사용됩니다.

#### HTTP의 Method 역할

HTTP 메소드는 클라이언트에서 서버로 전송되는 요청(request)의 종류를 나타내는 방법입니다.

HTTP 요청은 웹 요소(웹 페이지, 이미지, 동영상 등)를 요청하거나, 데이터를 서버로 전송할 때 사용됩니다. 이때, 요청에 따라 적절한 HTTP 메소드를 사용하여 서버와 클라이언트 간에 데이터를 전송하게 됩니다.

##### Method 종류

- GET: 웹 페이지 등을 요청할 때 사용되며, 요청한 정보를 받아오는 역할을 합니다.
- POST: 서버로 데이터를 전송할 때 사용되며, 데이터를 요청한 URL에 추가합니다.
- PUT: 서버에서 파일을 업로드할 때 사용되며, 요청한 URL에 파일을 추가합니다.
- DELETE: 서버에서 특정 데이터를 삭제할 때 사용되며, 요청한 URL에서 데이터를 삭제합니다.

리소드를 수정할 때 요청하는 메소드 중 PATCH와 PUT이 있는데 둘의 차이는
PATCH는 일부 수정, PUT은 전체 수정을 의미한다.
